# Clothing Showcaseinator
![alt text](assets/icon.png)
A tool intended for quickly creating showcases of classic (2D) Roblox clothing, so that you don't have to create them manually.

# How to use it
Simple.
1. Open it
2. Upload your shirt template and your pants template, upload manually or drag and drop the image
3. (Optional) Upload a "background". The intended size for the background is 704x274 pixels, but the uploaded image can stretch to fit
4. Save the output
